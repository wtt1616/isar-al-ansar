STEP 1: Install MySQL
----------------------

EASIEST METHOD - XAMPP (Recommended):

1. Download XAMPP: https://www.apachefriends.org/download.html
2. Install XAMPP (make sure to select MySQL + phpMyAdmin)
3. Open XAMPP Control Panel
4. Click "Start" next to MySQL
5. Double-click: setup-xampp.bat
6. Run: npm run dev
7. Open: http://localhost:3000

ALTERNATIVE - MySQL Server:

1. Download MySQL: https://dev.mysql.com/downloads/installer/
2. Install MySQL Server (remember the root password!)
3. Double-click: setup-mysql.bat
4. Edit .env file: set DB_PASSWORD=your_password
5. Run: npm run dev
6. Open: http://localhost:3000
